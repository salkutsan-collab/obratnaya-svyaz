# Обратная связь — дашборд

Интерактивный дашборд по результатам обратной связи двух программ
(Зимний университет, Проектный интенсив): оценки по дням, трекам, темам и
преподавателям, а также тематический анализ комментариев.

Страница: см. GitHub Pages этого репозитория (`index.html`).

Данные обезличены: имена участников в комментариях заменены инициалами,
преподаватели указаны инициалами.
